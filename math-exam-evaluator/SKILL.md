# 初中数学试卷评测技能

## 核心能力

1. **图片文字识别** — 识别试卷/题目图片中的文字、符号、图形
2. **答题对错判断** — 判断学生书写步骤是否正确，给出明确结论
3. **分步讲解** — 对错误步骤提供正确解法和思路点拨
4. **题目搜索** — 上网搜索类似题型供练习

## 支持学科范围

- **代数**：一元方程、二元一次方程组、不等式、因式分解、分式
- **几何**：三角形（全等/等腰/直角）、四边形、圆、相似形
- **函数**：一次函数、反比例函数、二次函数基础
- **统计与概率**：平均数、中位数、方差、概率基础

## 图片识别流程

当收到试卷/题目图片时：

```
Step 1：用 image 工具识别图片全部文字内容
Step 2：识别图形（如果有图，用 image 描述图形特征）
Step 3：对照学生答题步骤，逐行判断对错
Step 4：输出评测结果
```

## 评测结果格式

```markdown
## 📋 题目识别

**题目原文：**（从图片识别的题目文字）

**图形描述：**（如有配图，描述关键几何特征）

## ✅ 答题评测

| 步骤 | 学生答案 | 是否正确 | 说明 |
|------|----------|----------|------|
| 第1步 | ... | ✅ | 正确 |
| 第2步 | ... | ❌ | 错误：... |
| 第3步 | ... | ✅ | 正确 |

## 📝 总体评价

（整体思路是否正确，计算是否准确，逻辑是否清晰）

## 💡 正确解法

（分步写出正确解法，包含关键步骤说明）

## 🔍 类似练习题

（搜索1-2道同类练习题，供巩固）
```

## 题目搜索（辅助功能）

当需要搜索练习题时，使用 multi-search-engine：

```
搜索词格式：
- "初中数学 [知识点] 练习题"
- "初二几何 [题型] 中考题"
- "三角形全等 证明题 难题"

推荐引擎：百度（综合）+ 头条搜索（资讯）
```

## 注意事项

- 几何图形需重点描述形状、边长关系、角度、特殊点位置
- 如果图片模糊或内容不全，明确告知用户哪里无法识别
- 判断对错要明确指出错误原因，不只说"错了"而要说"哪里/为什么错"
- 涉及辅助线、复杂证明时，分步解释思路
