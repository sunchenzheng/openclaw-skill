## 新题相似度判断标准

### 判断维度
1. **题目条件相似度**：条件描述/已知数据相似程度
2. **解题方法相似度**：是否使用相同的几何模型或辅助线思路
3. **结论相似度**：题目问法和解题结论的相似程度

### 相似度阈值（高于此值则判定为重复，不予添加）
| 组合情况 | 阈值 |
|---------|------|
| ⭐ + ⭐⭐ 题（低难度）| 80% 相似度 → 替换已有题 |
| ⭐⭐⭐ 题（压轴）| 70% 相似度 → 不添加 |
| ⭐⭐⭐⭐ / ⭐⭐⭐⭐⭐ | 60% 相似度 → 不添加（鼓励变式创新）|

### 相似度处理流程
```
新题进来
  ↓
提取题目条件（角度/边长/模型类型）
  ↓
在同难度级别中查找相似题
  ↓
计算相似度评分
  ↓
若 ≥阈值 → 标记为"重复题"，记录来源，不添加
若 <阈值 → 审查题目质量 → 合格则添加至题库
```
