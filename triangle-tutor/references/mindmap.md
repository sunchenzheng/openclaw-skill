## 三角形全等判定方法对比

```
         ┌──────────────────────────────────────────┐
         │           全等三角形五大判定方法           │
         ├────────┬─────────┬────────────────────────┤
         │  方法  │  缩写   │         条件           │
         ├────────┼─────────┼────────────────────────┤
         │  边边边 │  SSS    │ 三条边分别相等         │
         │  边角边 │  SAS    │ 两条边 + 夹角相等      │
         │  角边角 │  ASA    │ 两个角 + 夹边相等      │
         │  角角边 │  AAS    │ 两个角 + 一条边相等    │
         │  斜边直角│  HL     │ 直角三角形的斜边+直角边│
         └────────┴─────────┴────────────────────────┘

  注意：SSA（边边角）不能判定全等！
```

## 等腰三角形 vs 等边三角形

```
           等腰三角形              等边三角形
              │                      │
       ┌──────┴──────┐          ┌─────┴─────┐
       │             │          │           │
     定义：        性质：        定义：      性质：
    两边相等      底角相等       三边相等    三角60°
       │             │           │           │
       │        ┌────┴────┐      │      ┌────┴────┐
    判定：      三线合一：      判定：     特殊等腰
  两角相等     顶角平分线       三边相等    （三边相等
  或两边相等   =底边中线       或三角60°    自动满足）
               =底边高
```
