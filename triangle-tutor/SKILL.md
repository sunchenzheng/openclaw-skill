# 三角形辅导技能

## 三角形知识体系

```
三角形
├── 基础概念
│   ├── 三角形定义（三边首尾相连的封闭图形）
│   ├── 三角形的分类（按边 / 按角）
│   ├── 三角形的重要线段（中线/高/角平分线/中位线）
│   └── 三角形内角和 = 180°
│
├── 全等三角形
│   ├── SSS（三边分别相等）
│   ├── SAS（两边及其夹角分别相等）
│   ├── ASA（两角及其夹边分别相等）
│   ├── AAS（两角及其中一角的对边分别相等）
│   └── HL（直角三角形的斜边和直角边）
│
├── 等腰三角形
│   ├── 定义：两边相等的三角形
│   ├── 性质：底角相等
│   ├── 判定：两角相等 或 两边相等
│   └── 三线合一：顶角平分线 = 底边中线 = 底边高
│
├── 等边三角形
│   ├── 定义：三边都相等
│   ├── 性质：三角均为60°
│   └── 判定：三边相等 或 三角均为60°
│
├── 直角三角形
│   ├── 勾股定理：a² + b² = c²
│   ├── 逆定理：用边长判断直角
│   ├── 30°角所对直角边 = 斜边一半
│   ├── 斜边中线 = 斜边一半
│   └── 射影定理
│
├── 相似三角形（扩展）
│   ├── SSS相似 / SAS相似 / AA相似
│   └── 性质：对应边成比例，对应角相等
│
└── 竞赛几何模型
    ├── 将军饮马模型
    ├── 胡不归模型
    ├── 阿氏圆模型
    ├── 费马点模型
    ├── 奔驰模型
    └── 12345模型
```

## 使用方式

### 学生问题目
1. 先判断题目涉及哪些知识点和难度星级
2. 查找对应题库（参考题号索引）
3. 如需要，分步引导学生思考而不是直接给答案

### 学生要求练习题
1. 了解学生当前学习进度和目标难度
2. 推荐难度合适的题目（建议从⭐开始，逐级递增）
3. 标记易错点，提前预防

### 学生问某个定理
1. 从知识框架中找到对应章节
2. 查看 `triangle_study_notes.md` 中的核心方法和常考题型
3. 给出1-2道对应的练习题

### 学生要求看视频学习
1. 查看 `video_library.md` 中的推荐路径
2. 根据难度定位推荐合适视频
3. 四周计划可作为长期学习指南

## 题目截图收集流程

详见 `references/screenshot_workflow.md`

### 优先级
1. **P1**：网上搜现成图片（百度图片/小红书/微信公众号/学科网）
2. **P2**：从视频截取题目完整展示帧
3. **P3**：从视频截取解答完整帧（翻页则多帧）

### 截图命名规范
```
q{编号}_question.png    # 题目截图
q{编号}_answer_1.png    # 解答截图（第一帧/页）
q{编号}_answer_2.png    # 解答截图（第二帧/页，如有翻页）
q{编号}_meta.json       # 元数据（来源BV号+时间戳）
```

### 截图工具
```bash
# 从视频截取单帧（需先下载视频到本地）
python scripts/screenshot_from_video.py <BV号> <题号> <题目时间戳> [解答时间戳1] [解答时间戳2] ...

# 示例
python scripts/screenshot_from_video.py BV16VVnz3EsU q091 00:12:34 00:15:20
```
