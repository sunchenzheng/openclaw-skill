# 差旅报销数据处理工具

## 快速开始

### 1. 安装依赖

```bash
pip install pandas openpyxl
```

### 2. 运行交互式程序

```bash
python interactive.py
```

### 3. 目录结构

```
C:\Users\123\openclaw_data\expense\
├── statements\     ← 放入银行流水 CSV 文件
├── invoices\       ← 放入数电发票 XML/PDF 文件
└── output\         ← 生成的报表输出目录
```

## 功能菜单

| 选项 | 功能 | 说明 |
|------|------|------|
| [1] 导入流水 | 从 CSV 导入账单 | 支持微信/支付宝/招行商务卡 |
| [2] 扫描发票 | 检查已收录发票 | 自动识别 XML/PDF 格式 |
| [3] 匹配发票 | 自动关联发票与流水 | 按金额+日期+商户名匹配 |
| [4] 标记垫款 | 确认大额公司垫款 | 超过 500 元自动提醒 |
| [5] 生成报表 | 导出报销核对清单 | Excel 多工作表格式 |
| [6] 查看状态 | 查看当前数据汇总 | 笔数/金额/报销进度 |
| [7] 使用指南 | 查看操作流程说明 | 首次使用必看 |

## 账单导出方式

### 微信支付
微信 → 我 → 服务 → 钱包 → 账单 → 右上角··· → 账单下载

### 支付宝
支付宝 → 我的 → 账单 → 右上角··· → 导出账单

### 招商银行商务卡(8651)
掌上生活 → 账单 → 已出账单 → 导出账单

## 消费类型

| 类型 | 规则 |
|------|------|
| 公司垫款 | 金额 ≥ 500 元，需发票报销 |
| 个人消费 | 自费，无需报销 |

## 报销状态

- ⏳ 未提交 — 有消费无发票
- ✅ 已开发票 — 已关联发票
- 💵 已报销 — 已完成报销

## 每月流程

每月初导出上月流水 → 发送 CSV 给 AI → 匹配发票 → 标记垫款 → 生成报表 → 上传发票云报销

## 技术说明

- Python 3.11+
- 依赖: pandas, openpyxl
- 交互式界面: 无需参数，直接运行
