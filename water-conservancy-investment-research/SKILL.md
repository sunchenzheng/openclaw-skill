# 水利基础设施建设投融资模式研究

本技能支持两类核心文档的撰写：

## 文档类型

### 1. 研究报告（Research Report）
框架结构：`references/research-report-framework.md`

典型任务：
- "写一篇水利基础设施建设投融资模式研究的研究报告"
- "分析当前水利投融资的主要模式及政策导向"
- "撰写某水利工程的投融资方案分析"

### 2. 银行调查报告（Bank Survey Report）
框架结构：`references/bank-survey-report-framework.md`

典型任务：
- "撰写鄂北水资源工程二期的银行调查报告"
- "写一份水利项目的银行融资调查报告"
- "为XX水利工程制作银行调查文档"

## 投融资模式参考

模式详解及对比：`references/financing-models.md`

主要模式：
- **政府投资**（中央/地方财政）
- **地方政府专项债券**
- **PPP模式**（政府和社会资本合作）
- **BOT/BOOT/BOO模式**
- **REITs**（不动产投资信托）
- **水权/水价改革融资**
- **开发性金融**（国开行、农发行）

## 政策参考

水利行业政策汇编：`references/policies.md`

关键政策：
- 《关于推进水利基础设施投资的指导意见》
- 《政府和社会资本合作建设重大水利工程》
- 水利领域不动产投资信托基金（REITs）试点政策
- 地方政府专项债相关政策

## 工作流程

### 研究报告流程
1. 确认报告侧重点（政策分析/模式对比/案例研究/建议）
2. 搜集相关政策文件和行业数据（使用 `scripts/search_water_policy.py`）
3. 确定报告结构（参考 `references/research-report-framework.md`）
4. 分章节撰写
5. 输出完整报告

### 银行调查报告流程
1. 确认项目基本信息（项目名称、建设内容、投资规模、进展阶段）
2. 搜集项目资料（可研批复、融资方案、合同协议等）
3. 确定调查报告结构（参考 `references/bank-survey-report-framework.md`）
4. 分章节撰写，重点关注：
   - 项目合规性（审批、用地、环境）
   - 融资方案（资金来源、资本金比例、还款来源）
   - 风险分析（政策风险、建设风险、运营风险、财务风险）
   - 还款能力测算
5. 输出完整调查报告

## 辅助脚本

### 搜索水利政策（search_water_policy.py）
```bash
python scripts/search_water_policy.py <关键词>
```
用途：搜索水利行业相关政策文件，返回政策名称、编号、发布日期、核心内容摘要。

### 获取项目信息（fetch_project_info.py）
```bash
python scripts/fetch_project_info.py <项目名称>
```
用途：联网搜索特定水利工程项目信息，包括项目概况、投资规模、进展阶段、中标单位等。

## 典型水利工程案例参考

- **鄂北水资源工程**（湖北）：引调水工程，政府投资+专项债
- **南水北调工程**：跨流域调水，中央投资为主
- **桃苑水务PPP项目**：社会资本参与
- **深圳水务集团REITs**：水利基础设施REITs试点

## 输出格式要求

- 语言：简体中文
- 标题层级：# 一级 / ## 二级 / ### 三级
- 段落：结构清晰，数据标注来源
- 表格：用于模式对比、参数对比
- 参考文献：标注政策来源和数据来源
